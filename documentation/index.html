Installation Instructions

Follow the instructions below to install Next.js for your hosting environment:

  1. Local Development -
    1. Open your terminal.
    2. Run yarn install to install dependencies.
    3. Run yarn dev to start the development server at http://localhost:3000.
  2. Deploying to Netlify - Watch Netlify Deployment Tutorial


  1. Local Build -
    1. Open your terminal.
    2. Run yarn build to generate the public directory.
    3. Run yarn start to start the local server at http://localhost:3000.
  2. Deploying to Vercel - Watch Vercel Deployment Tutorial


Other Scripts
  1. Open your terminal.
  2. Run yarn lint:fix to format code and fix auto-fixable ESLint errors.
